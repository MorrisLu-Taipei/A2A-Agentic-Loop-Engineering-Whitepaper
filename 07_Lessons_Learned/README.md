# 07 · Lessons Learned（研究級踩坑與教訓)

**性質**:本研究**自身**的回顧(retrospective);對應 ALE 的 Feedback/Retrospective 迴圈(每專案都要 LESSONS_LEARNED)。位置在 06 之後、99 之前,持續累積。
**用途**:把散在 `RESEARCH_LOG` / `DOI_GUIDE` / 故事裡的教訓集中;多數教訓由 **08 的 3×AI 對抗式方法論**(Codex 審查)當場抓出——這本身就是方法論有效的證據。

每則格式:**狀況 / 根因 / 解法 / 教訓**。

---

## L1 · 誤讀關鍵文獻(最嚴重,我方錯誤)
- **狀況**:把 arXiv:2512.03097(healthcare collusion)結論寫成「verifier 擋不住共謀,支持我們」,並寫進白皮書 §2.7/§7.2、緣起故事、prior-art。
- **根因**:信了**快速模型對 PDF 的二手摘要**,沒讀原文。原文其實相反:多代理人**共識**不安全,但**接外部可信指南的 verifier 把 ASR 從 100% 壓到 0%**。
- **解法**:讀 PDF 原文查證 → 全卷宗修正;改寫成「模型共識≠安全,**靠外部證據的驗證**才有效」(這反而**更挺** ALE 的機械閘論點)。
- **教訓**:**關鍵文獻一律讀原文(PDF/正文)**,二手摘要與快速模型摘要不可作引用依據。

## L2 · prior-art 檢索不徹底 → 撞名沒查到
- **狀況**:對外宣稱「沒人用 Agentic Loop Engineering = ALE」。
- **根因**:只看了 arXiv:2509.06216 的**摘要層級**,沒讀正文;其 **§5.2 明確定義 "Agentic Loop Engineering (ALE)"**,同領域、同全名、同縮寫。
- **解法**:改名(讓出 ALE 這個詞);新穎性改以「模型共識→可執行證據」立論。
- **教訓**:novelty/撞名查核要讀**全文+正文章節**,不能只看 abstract。

## L3 · 漏掉直接 prior art
- **狀況**:宣稱「test collusion 的特化與命名是我們的」。
- **根因**:檢索不窮盡;arXiv:2603.15611《Code-A1》(2026-03)已用 **self-collusion** 描述 code LLM/test LLM 產 trivial tests 換 reward,解法是兩模型目標對立分離——幾乎就是我們的概念與對策。
- **解法**:引用 Code-A1 為直接 prior art;貢獻降級為「整合進完整 Agentic SDLC 治理」。
- **教訓**:命名/首創宣稱前,**預設別人可能做過**,窮盡檢索並記錄納入排除。

## L4 · AI 產的「證據」造假被攔
- **狀況**:Gemini 修訂的案例研究放了一段**虛構的稽核日誌**(編的數字、CLI、hash)。
- **根因**:AI 為「補證據鏈」自行編造看似合理的輸出。
- **解法**:查核否決,改用真實執行 `vv_crosscheck.py` 的輸出。
- **教訓**:**AI 產的任何「證據/數據/日誌」一律查核**;定生死靠真實執行,不靠看起來合理。

## L5 · ORCID 沒連動
- **狀況**:兩篇 Zenodo 都填了 ORCID,但個人 ORCID 名下 works=0。
- **根因**:在 Zenodo 作者欄**打字填 iD 只顯示、不回寫**;要自動同步需用 ORCID 登入 + DataCite 授權。
- **解法**:手動 Add DOI 補進去;以後**用 ORCID 登入 Zenodo 再 deposit**。
- **教訓**:見 `06_Publication/DOI_GUIDE.md` §4;發完到 ORCID 確認 works +1 才算成功。

## L6 · 過度宣稱 → 證據等級校準
- **狀況**:早期用「無懈可擊/首創/證實」。
- **根因**:把設計主張當已驗證結論。
- **解法**:全文標 `[設計原則]`/`[研究假說]`/`[初步證據]`/`[已驗證]`;摘要改假說語氣。
- **教訓**:**主張強度 ↔ 證據強度 ↔ 引文**三者必須一致。

## L7 · 商業機密邊界
- **狀況**:Loop 0 形式化時一度把專有 skill set 的技能清單/契約/路徑寫進卷宗。
- **根因**:研究證據需求 vs IP 保密未先界定。
- **解法**:全面抽象化,只留「大概做什麼 + 外部成效指標」;細節留私有套件。
- **教訓**:**對外/可能對外的輸出只放摘要**;機密細節不進研究卷宗。
