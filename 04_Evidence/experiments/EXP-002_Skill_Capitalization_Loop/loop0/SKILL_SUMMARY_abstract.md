# Loop 0 · 萃取 Skill 摘要（抽象層級;細節為商業機密)

> 🔒 **機密界線**:本案萃取出的交付 skill set 是 TigerAI **商業機密**。本檔**只說明它大概做什麼、以及在 ALE 研究中的角色**;**不列出**技能清單、資料合約內容、內部結構、檔案路徑或實作。完整內容僅存於私有 skill 套件(不在本研究卷宗、不隨論文/DOI 釋出)。

## 這個 Skill 大概做什麼(可公開的抽象描述)
- 一套 **TigerAI 專有的「資料 → 自動化儀表板」交付 skill set**。
- 覆蓋**完整 SDLC 階段**:資料理解 → 規劃 → 設計 → 施工 → **V&V(自帶驗證關卡)** → 資安 → 部署 → 監控 → 結案;自動化以工作流引擎為主幹。
- **可驗證性內建**:以「資料契約 + 系統級不變量機械閘 + smoke」作為交付前的客觀關卡(對應 ALE §7.3/§7.7)。
- 設計為**可重用、可組裝**:新案以配置/組裝為主,儘量不重寫程式。

> 以上為抽象說明;**「具體有哪些技能模組、各自怎麼做、契約欄位為何」屬機密,不在此記錄。**
