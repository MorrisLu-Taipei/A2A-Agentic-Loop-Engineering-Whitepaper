# 論文各章寫作檢查清單(方法論可重用資產)

> 用途:`08_Methodology` 的寫作品質清單,供 producer(Claude)寫、reviewer(Codex)審時逐項對照。**用我們自己的話整理**,原則來源附於每節末(尊重版權,不照抄外部文字)。
> 目前涵蓋:**Discussion**。日後可續加 Abstract / Introduction / Related Work / Methods / Results。

## A. Discussion(討論)章 — 檢查清單

### 結構(建議六步)
1. **開場點出研究缺口**:本研究要補的是哪個 gap(呼應 Introduction,不要重講一遍)。
2. **摘要關鍵發現 + 佐證**:用最關鍵的數據/結果支撐主結論(不是把 Results 全抄一次)。
3. **放回文獻脈絡解讀**:與同儕審查文獻比較——相符?衝突?延伸?(這是 Discussion 的靈魂,Results 沒有的「為什麼重要」)。
4. **誠實列限制與其影響**:樣本小、雜訊、效應弱、混淆變項;說清楚限制如何影響可推論性。
5. **實務與未來研究意涵**:對實務有何用;留下哪些未解問題、下一步怎麼做。
6. **收尾回到原研究問題**:結論重述核心發現,扣回開頭的 RQ。

### 要做(DO)
- [ ] 只挑**支撐主結論**的數據來談,不是 Results 的複述。
- [ ] **對照已發表文獻**(相符/衝突/延伸),不要孤立陳述。
- [ ] 講**實務相關性**(practical relevance)。
- [ ] 點出**本研究未解的知識缺口**。
- [ ] 把發現**連回更大的研究地景**。
- [ ] 結尾**重述核心發現**並扣回 RQ。

### 要避免(AVOID)
- [ ] ❌ 只重述結果、沒有新解讀。
- [ ] ❌ 在 Discussion **冒出 Results 沒有的新數據/新發現**。
- [ ] ❌ **過度宣稱**(超出數據能支撐的程度)。
- [ ] ❌ 不提限制。
- [ ] ❌ 不連結既有文獻。
- [ ] ❌ 偏離原研究問題。
- [ ] ❌ 段落鬆散、與證據脫節。

## B. 套用到 ALE 白皮書 §12(現況落差,供修訂參考)

ALE 是 **Working Paper、尚無 [validated] 結果**,所以「以結果為核心」的 Discussion 只能部分套用;但仍可校準:
- ✅ 已有:§12.2 限制與威脅效度(對應步驟 4)、§12.3 倫理責任(對應實務/責任意涵)。
- ⚠ 可補:§12 開頭**點出研究缺口**(步驟 1)、**扣回 RQ1–RQ5 收尾**(步驟 6)、把「與文獻比較」從 §2.7 呼應進 §12(步驟 3)。
- ⚠ 注意紅線:本研究**沒有實證結果**,Discussion 不可寫成「已證明」——維持 `[研究假說]/[初步證據]` 語氣(與 §9.7 Claim–Evidence Matrix 一致)。
